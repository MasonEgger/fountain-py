Core Modules
------------

The fountain-py library consists of four main modules:

* **Parser**: The :doc:`parser` module handles parsing Fountain text into structured elements
* **Document**: The :doc:`document` module provides the main document container with analysis methods
* **Elements**: The :doc:`elements` module defines all Fountain element types and formatting
* **Renderer**: The :doc:`renderer` module converts documents to different output formats
